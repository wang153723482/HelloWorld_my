# HelloWorld_gatling

demos of gatling script.  
gatling的脚本例子。

==OK==

s1_vu  setting number of vu.  
设置虚拟用户的数量

s2_rampup  setting start up time to new vu.  
设置虚拟用户的启动时间

s3_duration  the script running time.  
设置运行时间

s4_post 

post简单的key=value参数


s5_req_json  
post请求中body的内容，可以是json或xml，也可以是其他的东西

==NG==


param_csv  
csv参数化

assert  
断言

response reg    
正则解析

responese json  
json解析
